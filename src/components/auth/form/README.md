# Form Components

Folder ini berisi komponen-komponen form yang digunakan dalam aplikasi Money Tracker.

## Komponen

- `FormCheckbox.jsx` - Komponen checkbox untuk form
- `FormContainer.jsx` - Komponen container untuk form
- `FormDatePicker.jsx` - Komponen date picker untuk form
- `FormFile.jsx` - Komponen upload file untuk form
- `FormGroup.jsx` - Komponen group untuk mengelompokkan elemen form
- `FormInput.jsx` - Komponen input text untuk form
- `FormRadioGroup.jsx` - Komponen radio button group untuk form
- `FormRange.jsx` - Komponen slider/range untuk form
- `FormRow.jsx` - Komponen row untuk layout form
- `FormSelect.jsx` - Komponen dropdown select untuk form
- `FormSwitch.jsx` - Komponen toggle switch untuk form
- `FormTextarea.jsx` - Komponen textarea untuk form

## Penggunaan

Komponen-komponen ini dapat diimpor dari `../components/auth` melalui file index.js.

```jsx
import { FormInput, FormGroup, FormContainer } from '../components/auth';
```
