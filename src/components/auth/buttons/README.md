# Button Components

Folder ini berisi komponen-komponen button yang digunakan dalam aplikasi Money Tracker.

## Komponen

- `IconButton.jsx` - Komponen button dengan ikon
- `LinkButton.jsx` - Komponen button yang berfungsi sebagai link
- `OutlineButton.jsx` - Komponen button dengan style outline
- `PrimaryButton.jsx` - Komponen button utama dengan style solid
- `SubmitButton.jsx` - Komponen button khusus untuk submit form

## Penggunaan

Komponen-komponen ini dapat diimpor dari `../components/auth` melalui file index.js.

```jsx
import { SubmitButton, LinkButton, PrimaryButton } from '../components/auth';
```
